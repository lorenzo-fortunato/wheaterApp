Applicazione creata con l'utilizzo di JavaScript Vanilla.

E' stata utilizzata l'API free di openweathermap (https://home.openweathermap.org).

Per il funzionamento dell'app in locale è necessario andare su https://home.openweathermap.org, creare un account se non lo si possiede già e generare una key.

Dopodiché bisogna andare nel file index.js ed istanziare:
const apiKey = "la_tua_key";
inserendo il valore generato.

Ignorare il commento a riga 42 nel file index.html, necessario per valorizzare correttamente la mia variabile apiKey.
